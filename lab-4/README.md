# Лабораторна робота №4: Візуалізація даних 2

### Мета роботи
Створення інтерактивної програми для моделювання та візуалізації гармонічного сигналу $y(t) = A \cdot \sin(\omega t + \phi)$ з можливістю накладання шуму та його подальшої фільтрації.

### Основні функції
- **Інтерактивне керування**: Зміна амплітуди, частоти, фази сигналу та параметрів шуму (середнє значення, дисперсія) за допомогою слайдерів.
- **Гнучкість**: Можливість увімкнути/вимкнути відображення шуму.
- **Фільтрація**: Використання IIR-фільтра (Баттерворта) для очищення сигналу від шуму в реальному часі.
- **Reset**: Швидке повернення до початкових налаштувань.
- **Оптимізація**: Збереження стану шуму для запобігання зайвим обчисленням при зміні параметрів гармоніки.
