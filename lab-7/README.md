# Лабораторна робота №7: Face Detection та підрахунок FPS

## Мета роботи
Розширення функціоналу попередньої роботи, інтеграція DNN-моделі для розпізнавання облич та оптимізація відображення.

## Функціонал
* Включає всі можливості Лабораторної роботи №6.
* **Face Detection:** Детекція облич у реальному часі за допомогою моделі Caffe (SSD). Вмикається/вимикається клавішею `f` або `F`.
* **Лічильник FPS:** Динамічне обчислення та виведення частоти кадрів на екран.
* Коректне накладання кольорових рамок (зелений квадрат) поверх чорно-білих режимів (Grayscale, Canny Edge).

## Вимоги до моделі
Для роботи детектора облич необхідні файли у робочій директорії (`build`):
* `deploy.prototxt`
* `res10_300x300_ssd_iter_140000.caffemodel`
