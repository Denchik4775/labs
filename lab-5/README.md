#  Лабораторна робота №5

##  Мета роботи
Створення інтерактивного веб-додатка для аналізу та візуалізації даних вегетаційного індексу (VHI) з використанням бібліотеки **Streamlit**.

---

##  Основні функції
- **Інтерактивне керування**: Вибір типу індексу (VCI, TCI, VHI) та регіону через dropdown-списки.
- **Гнучка фільтрація**: Використання слайдерів для відбору даних за заданими інтервалами років та тижнів.
- **Візуалізація**:
    - **Таблиця**: Відображення відфільтрованих даних у зручному форматі.
    - **Графік 1**: Побудова часових рядів для обраного діапазону.
    - **Графік 2**: Порівняльний аналіз обраної області з іншими регіонами.
- **Сортування**: Додано чекбокси для сортування даних за зростанням/спаданням обраного показника.
- **Reset**: Швидке повернення інтерфейсу та фільтрів до початкових значень.

---

##  Як запустити
1. **Завантаження даних**: Запустіть скрипт `downloader.py` для отримання актуальних даних NOAA.
2. **Запуск додатку**: Використайте команду `streamlit run app.py` у папці проекту.
3. **Робота з даними**: Оберіть параметри на бічній панелі та перемикайтеся між вкладками для аналізу результатів.
