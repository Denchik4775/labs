# Лабораторна робота №3: Візуалізація даних

### Опис завдання
Цей репозиторій містить виконання лабораторної роботи №3 з візуалізації багатовимірних даних. 

Для аналізу було обрано **Automobile Data Set** з UCI Machine Learning Repository. Датасет повністю відповідає вимогам: має багатовимірну структуру (Multivariate), містить категоріальні, цілі та дійсні атрибути, а також містить пропущені значення.

### Виконані завдання:
1. Здійснено очищення даних (Data Cleaning): оброблено нестандартні символи пропусків (`?`), приведено атрибути до числових типів та заповнено відсутні значення (медіаною та модою).
2. Побудовано графік залежності дійсних атрибутів (Scatter plot залежності ціни від витрати пального).
3. Побудовано гістограму розподілу потужності двигунів (кінських сил) рівно у 5 заданих діапазонах.
4. Створено складні графіки для ефективної візуалізації понад двох вимірів (Scatter plot із параметрами hue та size, Boxplot за двома категоріями, Heatmap кореляції, Pairplot). Загальна кількість побудованих графіків — 6.

### Інструкція із запуску
1. Відкрийте термінал у корені репозиторію та перейдіть у гілку `lab3`: `git checkout lab3`
2. Переконайтеся, що активовано віртуальне середовище.
3. Встановіть необхідні залежності: `pip install pandas numpy matplotlib seaborn`
4. Запустіть Jupyter Notebook: `jupyter notebook`
5. Виконайте комірки у файлі `lab3.ipynb` по черзі. Датасет завантажується за посиланням автоматично.
